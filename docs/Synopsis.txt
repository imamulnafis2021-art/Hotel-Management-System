The Starlight Hotel Management System is a comprehensive
console-based application developed in C++17 that automates
the day-to-day operations of a modern hotel. The project
demonstrates mastery of Object-Oriented Programming through
14 implementation phases covering OOP design patterns,
file persistence, authentication, STL algorithms, regex
validation, and modern C++ features.

13. Smart Recommendation  - Budget + Guest Matching
